
# TikTok Ball Game

A Simple Python Game Inspired By The Popular TikTok Bouncing Ball Game 

## Installation:

```bash
  git clone https://github.com/Kyle8973/TikTok-BallGame
  cd TikTok-BallGame
  python3 ballgame.py
```
    
## Usage:

```py
python3 ballgame.py
```
